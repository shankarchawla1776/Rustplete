# rusty-notes

A project to function as a notetaking app for your terminal using rustyline and chrono. 

### Usage 

1. Clone the repository to your local machine. 

2. Navigate to the project and run it with cargo. 
```sh
cd rusty-notes
cargo build --release 
cargo run
```
